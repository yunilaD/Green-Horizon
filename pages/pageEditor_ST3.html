Feedback page

Technical Description

The Feedback Page was designed using HTML and CSS with structured layout to ensure clarity and usability. The page consists of a form tag that collects user inputs through various fields. The form submission is handled via a 3rd party API(https://api.web3forms.com/submit)The CSS applies a clean and modern aesthetic using the Poppins font and a decorative background to make the page visually appealing.

Accessibility

  • Keyboard Navigation: All the fields and buttons on the Feedback Page can be accessed using the keyboard.
  • Alternative Text For Images: Each image includes an alt attribute to describe its content, ensuring accessibility for visually impaired users.

Team Page

Technical Description

The Team Page employs a card-based layout for presenting team members, where each card contains an image, name and a list of contributions. The team.css file uses a flex box for responsive alignment and hover effects add interactivity.

Accessibility

  • Alternative Text For Images: Each image includes an alt attribute to describe its content, ensuring accessibility for visually impaired users.
  • Contrast and Readability: The Team Page employs a semi-transparent overlay on images to improve text readability and maintains a strong contrast ratio.

Challenges and Lessons Learned

  • Hover Effects & Interactivity: Implementing smooth animations without performance lag was a challenge. CSS animations and cubic-bezier easing were optimized for better performance.
  • Form Submission Handling: Initially, the Feedback Page faced issues with incorrect API integration. This was resolved by ensuring the correct access key was passed.
  • Testing with Users: Conducting basic usability tests helped refine the design for clarity

Compliance

  • HTML5 & CSS3 Standards: The pages pass W3C validation with minor warnings addressed.
  • GDPR Compliance: The Feedback Form does not store data locally and relies on a secure API.
